### DOM Events Used in This Code — Definitions + What They Do
### 1. click Event

#### Definition:
Triggers when an element is clicked with the mouse.

#### In your code:
When the button is clicked, the .changeColor div changes to pink and shows the text "button is clicked".

### 2. mouseover Event

#### Definition:
Occurs when the mouse pointer enters (hovers over) an element.

#### In your code:

On the .changeColor div → it increases font size, adds border, adds padding.

On the image → it changes the picture to a new image.

### 3. mouseout Event

#### Definition:
Occurs when the mouse leaves an element.

#### In your code:

On the .changeColor div → it resets the font, border, and padding back to normal.

On the image → it switches the picture back to the original one.

### 4. mousedown Event

#### Definition:
Fires when a mouse button is pressed down on an element (before releasing).

#### In your code:
When the user presses and holds the button, the text becomes "mouse button press".

### 5. mouseup Event

#### Definition:
Fires when the mouse button is released.

#### In your code:
When the button is released, the text changes to "button leave".

### 6. keydown Event

#### Definition:
Occurs when a key on the keyboard is pressed down.

#### In your code:
As the user types in the input box, the text being typed appears live in the .inputtext div.
